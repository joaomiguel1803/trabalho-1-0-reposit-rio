Visite o site da OpenAI Pesquise e escreva 1 parágrafo explicando o que é o GitHub Pages. O GitHub Pages é um serviço gratuito oferecido pelo GitHub que permite criar e hospedar sites estáticos diretamente a partir de repositórios Git. Ele é ideal para publicar páginas pessoais, blogs ou a documentação de projetos. Basta enviar os arquivos HTML, CSS e JavaScript para um repositório e, em seguida, ativar a funcionalidade no GitHub para gerar uma URL onde o site ficará acessível online. É uma maneira simples e eficiente de compartilhar conteúdo sem precisar de servidores externos.
